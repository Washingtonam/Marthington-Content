# Component-Driven Prompt Plan

Use these prompts to expand the content factory safely and repeatedly.

## 1. Backend content model
Prompt:
Create a MongoDB model for content items with fields for title, slug, category, summary, affiliateUrl, publishedAt, and tags. Add a CRUD route layer for managing these items.

## 2. Frontend content list
Prompt:
Create a reusable React component that displays a list of content cards with title, summary, category, and affiliate CTA. Connect it to the backend API.

## 3. SEO page template
Prompt:
Create a content detail page component with SEO-friendly metadata, a hero section, and affiliate CTA blocks.

## 4. Admin dashboard
Prompt:
Create an admin dashboard view that supports creating, editing, and deleting content items through the API.

## 5. Niche adapter
Prompt:
Create a configuration layer that maps this template to a specific niche such as travel, fitness, finance, or SaaS.
